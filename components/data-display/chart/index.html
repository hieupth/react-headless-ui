Chart

Lightweight SVG charts backed by the headless useChart hook. It draws line, bar, area, pie, and scatter charts from one or more datasets, computing scales, axes, legends, and optional tooltips. Because it renders inline SVG it appears unstyled-by-CSS — set dimensions and colors through props.

Line chart

Each dataset is { label, data, color } where data points are { x, y }.

Preview

Switch type to bar for categorical values.

NameTypeDefaultDescription
type'line' | 'bar' | 'pie' | 'area' | 'scatter''line'Chart kind.
datasetsChartDataset[]—Series: { label, data: {x,y}[], color?, fill? }.
width / heightnumber—SVG dimensions.

legendChartLegend—Legend visibility and placement.
colorsstring[]—Override the default palette.
responsiveboolean—Scale to the container width.
animated / animationDurationboolean / number—Enter animation controls.
showTooltipsboolean—Render data-point tooltips.
onDataPointClick(dataset, point) => void—Point click handler.
pointRenderer / barRenderer / lineRenderer(…) => ReactNode—Replace the default SVG primitives.
